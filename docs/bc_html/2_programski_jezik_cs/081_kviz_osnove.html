Квиз¶

Питање 1¶

Ако се говори о методи Main() у класи Program, означи исказ који НИЈЕ тачан.

Може да има низ стрингова args као једини параметар.

Не мора да има никакав параметар.

Не мора да буде статичка метода.

Може да буде типа void или типа int.

Питање 2¶

Штиклирај тачне исказе.

Логичка неистина представља се нулом, а логичка истина бројем различитим од нуле.

Знаковним типом представљају се 16-битни Unicode UTF-16 карактери.

Целобројне вредности представљају се типом int са или без модификатора.

Могуће је вршити конверзије између било која два нумеричка типа.

Објекти референтних типова се чувају на хипу.

Питање 3¶

Шта ће се исписати у конзоли након извршења следећег програма?

using System;

class Program
{
    static void Main()
    {
        double res = 9 + 8 * 7 - 6 / 4;
        Console.WriteLine(res);
    }
}

Означи тачан одговор.

64.0

64

4.0

4

Питање 5¶

Под претпоставком да ће корисник унети цео број, шта ће се исписати у конзоли након извршења следећег програма?

using System;

class Program
{
    static void Main()
    {
        int a = int.Parse(Console.ReadLine());
        int b = 0;
        while (a != 0)
        {
            b += a % 10;
            a /= 10;
        }
        Console.WriteLine(b);
    }
}

Означи тачан одговор.

Прва цифра унетог целог броја.

Збир цифара унетог целог броја.

Цифре унетог целог броја у обратном распореду.

Последња цифра унетог целог броја.
