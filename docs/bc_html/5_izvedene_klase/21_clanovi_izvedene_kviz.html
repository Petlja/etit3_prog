Питање 2¶

Дефинисане су три класе:

class Program
{
    public static void Main(string[] args)
    {
        B b = new B();
        b.Metod(5);
        Console.WriteLine("b.i je " + b.CitajI());
    }
}

class A
{
    int i;

    public int CitajI()
    {
        return i;
    }

    public void Metod(int i)
    {
        this.i = i;
    }
}

class B : A
{
    public void Metod(string s)
    {
        Console.WriteLine(s);
    }
}

Анализирај дати кôд и одреди да ли је кôд исправно написан. Означи исказ који даје информацију о тачности кода.

Програм има грешку, јер је метод Metod(int i) надјачан (предефинисан) са различитим потписом у класи B.

Програм има грешку, јер се b.Mетод(5) не може позвати пошто је метод Metod(int i) заклоњен у класи B.

Програм има грешку због b.i, јер је поље i неприступачно из класе B.

Програм нема грешке, јер наслеђени метод класе А, Metod(int i) није надјачан у класи B, већ је дефинисан преоптерећен метод Metod(string s).

Питање 5¶

Дефинисане су две класе и класа Program са методом Main().

using System;
class A
{
    private int n = 10;
    public class B : A
    {
        private int m = 20;
        public void Print()
        {
            Console.WriteLine($"{n},{m}");
        }
    }
}
class Program
{
    static void Main()
    {
        A.B b = new A.B();
        b.Print();
    }
}

Штиклирај тачне исказе

Програм се неће компајлирати.

Програм ће се успешно компајлирати и извршити.

У изведеној класи B доступно је приватно поље n базне класе А.

У изведеној класи B није доступно приватно поље n базне класе А, јер се приватни чланови не наслеђују.

Није могуће инстанцирати објекат b јер за класу A није експлицитно наведен модификатор приступа.

У конзоли ће се исписати 10,20.
