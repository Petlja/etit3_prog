Питање 3¶

Дефинисане су три класе:

namespace TestPrimer
{
    class Program
    {
        static void Main(string[] args)
        {
            KlasaB b = new KlasaB();
            b.Print();
        }
    }

    class KlasaA
    {
        string s;
        public KlasaA(string s) { this.s = s; }
        public void Print() { Console.WriteLine(this.s); }
    }

    class KlasaB : KlasaA { }
}

Анализирај дате класе и процени који су од понуђених исказа тачни. Штиклирај очекиване одговоре.

Програм има грешку, јер KlasaB нема подразумевани конструктор KlasaB().

Програм има грешку јер KlasaB има подразумевани конструктор, док родитељска KlasaА нема такав конструктор. Програм би радио без грешке уколико би се уклонио конструктор са параметрима из KlasaА.

Програм има грешку која се може отклонити уколико би се у KlasaА експлицитно додао конструктор без параметара KlasaА().

Програм нема грешку, извршава се, али се на конзоли ништа не исписује јер је поље s добило подразумевану вредност String.Empty.

Питање 4¶

Нека базна класа има дефинисан јавни подразумевани конструктор…

class BaznaKlasa
{
    public BaznaKlasa() { }
}

…и нека је у изведеној класи дефинисан јавни подразумевани конструктор:

class IzvedenaKlasa : BaznaKlasa
{
    public IzvedenaKlasa() { }
}

Означи тачан исказ.

Доћиће до грешке јер у базној класи није експлицитно позван јавни подразумевани конструктор класе System.Object.

Доћиће до грешке јер у изведеној класи није експлицитно позван јавни подразумевани конструктор базне класе.

Неће доћи до грешке јер ће компајлер сам допунити кôд конструктора у изведеној класи кључном речју base.

Неће доћи до грешке јер се увек прво позива конструктор изведене класе.

Питање 5¶

Означи тачан исказ.

Класа System.Object нема јавни подразумевани конструктор.

Класа System.Object има јавни подразумевани конструктор који се мора експлицитно позвати кључном речју base.

Класа System.Object има јавни подразумевани конструктор који се не мора експлицитно позивати.

Из класе System.Object се не наслеђују друге класе, па позив њеног јавног подразумеваног конструктора није битан.
