Квиз¶

Питање 1¶

Који су од наведених исказа тачни?

Апстрактне класе су класе које не могу бити инстанциране.

Запечаћене класе су класе које не могу бити инстанциране.

Из апстрактне класе не могу се изводити друге класе.

Из запечаћене класе не могу се изводити друге класе.

Питање 2¶

У којим случајевима ће компајлер јавити грешку током компајлирања?

У класи која је изведена из апстрактне, нису имплементиране све методе апстрактне класе.

У апстрактној класи дефинисана је неапстрактна метода.

Апстрактна класа је изведена је из апстрактне класе.

Креирана је инстанца апстрактне класе.

Питање 3¶

Који су од наведених исказа тачни ако се говори о апстрактним класама?

Апстрактна класа не сме да садржи поља.

Апстрактна класа не мора да садржи апстрактне чланове.

Сви чланови апстрактних класа морају бити апстрактни.

Апстрактна класа може да садржи и апстрактне и неапстрактне чланове.

Питање 4¶

Који су од наведених исказа тачни ако се говори о запечаћеној класи?

Запечаћена класа не може се користити као базна класа.

Запечаћена класа мора да садржи запечаћене чланове.

Из запечаћене класе не се могу изводити друге класе.

Запечаћена класа не може бити изведена из друге класе.

Питање 5¶

Који су од наведених исказа тачни, ако се говори о запечаћеним методама?

Могуће је запечатити само методу која преоптерећује виртуелну методу.

Када се једном метода запечати, више се не може преоптеретити.

Запечаћивање метода негативно утиче на перформансе програма.

Ако класа има запечаћену методу, онда се из ње не могу изводити друге класе.

Питање 6¶

Једна од основних особина објектно-оријентисаног језика је наслеђивање. Дате су насловне линије дефиниције класе. Означи дефиницију класе која се НЕ може наследити.

class A { }

class A { private A(){ } }

sealed class A { }

class A { protected A(){ } }

Питање 7¶

Да би наслеђени метод могао да се редефинише и тиме измени његова функционалност у изведеним класама, у базној класи испред ознаке повратног типа метода наводи се нека од понуђених кључних речи. Штиклирај кључне речи које омогућавају редефинисање дефинисаног метода кроз ланац наслеђивања.
