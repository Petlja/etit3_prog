## Питање 3

Дефинисане су три класе:

```cs
namespace TestPrimer
{
    class Program
    {
        static void Main(string[] args)
        {
            KlasaB b = new KlasaB();
            b.Print();
        }
    }

    class KlasaA
    {
        string s;
        public KlasaA(string s) { this.s = s; }
        public void Print() { Console.WriteLine(this.s); }
    }

    class KlasaB : KlasaA { }
}
```

```{mchoice}
:answer1: Програм има грешку, јер `KlasaB` нема подразумевани конструктор `KlasaB()`.
:answer2: Програм има грешку јер `KlasaB` има подразумевани конструктор, док родитељска `KlasaА` нема такав конструктор. Програм би радио без грешке уколико би се уклонио конструктор са параметрима из `KlasaА`.
:answer3: Програм има грешку која се може отклонити уколико би се у `KlasaА` експлицитно додао конструктор без параметара `KlasaА()`.
:answer4: Програм нема грешку, извршава се, али се на конзоли ништа не исписује јер је поље `s` добило подразумевану вредност `String.Empty`.
:correct: 2,3

Анализирај дате класе и процени који су од понуђених исказа тачни. Штиклирај
очекиване одговоре.
```

## Питање 4

Нека базна класа има дефинисан јавни подразумевани конструктор...

```cs
class BaznaKlasa
{
    public BaznaKlasa() { }
}
```

...и нека је у изведеној класи дефинисан јавни подразумевани конструктор:

```cs
class IzvedenaKlasa : BaznaKlasa
{
    public IzvedenaKlasa() { }
}
```

```{mchoice}
:answer1: Доћиће до грешке јер у базној класи није експлицитно позван јавни подразумевани конструктор класе `System.Object`.
:answer2: Доћиће до грешке јер у изведеној класи није експлицитно позван јавни подразумевани конструктор базне класе.
:answer3: Неће доћи до грешке јер ће компајлер сам допунити кôд конструктора у изведеној класи кључном речју `base`.
:answer4: Неће доћи до грешке јер се увек прво позива конструктор изведене класе.
:correct: 3

Означи тачан исказ.
```

## Питање 5

```{mchoice}
:answer1: Класа `System.Object` нема јавни подразумевани конструктор.
:answer2: Класа `System.Object` има јавни подразумевани конструктор који се мора експлицитно позвати кључном речју `base`.
:answer3: Класа `System.Object` има јавни подразумевани конструктор који се не мора експлицитно позивати.
:answer4: Из класе `System.Object` се не наслеђују друге класе, па позив њеног јавног подразумеваног конструктора није битан.
:correct: 3

```
